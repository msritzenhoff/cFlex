cFlex
Shape and display with Flexbox

A kind of blueprint/framework to setup a website layout for modern browser with CSS Flexbox.

Along with the new release, a separate base file [cflex3_radix.css] for the faster start of layouts has been included. Based on the practical experience, this provides a minimal default setting. The ingredients make the two files larger. If you minify them, unused/commented options will be removed and very short loading times will be achieved.

Integrate cFlex into your static or dynamic website

All starts with adding the style

<link rel="stylesheet" href="./<PATH_TO_CSS>/normalize.css?v=8.0.1" />
<link rel="stylesheet" href="./<PATH_TO_CSS>/cflex3.css?v=3.0.4" />
<link rel="stylesheet" href="./<PATH_TO_CSS>/cflex3_radix.css?v=3.0.4" />



cflex3.css file has a weight at 6,4 kB / minified 1,7 kB

The cflex3.css file is the main framework which can be used as default.

cflex3_radix.css file has a weight at 7,7 kB / minified 3,0 kB

The cflex3_radix.css file is a basic template, with a few basic functions to get started quickly.
