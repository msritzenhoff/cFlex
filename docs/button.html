Button

Only flat styled

A link is a link and a button is button.
…but if needed, there is a class to let a link shine like a button


Single Button

button, a.cf_btn,
button.btnRound, a.btnRound,
button:visited, a.cf_btn:visited,
button.btnRound:visited, a.btnRound:visited{
  width:auto; padding:5.45px 1.42em;
  background:#e0e0e0;
  border:1px solid #a1a1a1;
}
button.btnRound:first-of-type,
a.btnRound:first-of-type{
  border-radius:4px;
}
a.cf_btn{
  padding:5.45px 1.50em;
}

button:hover, a.cf_btn:hover,
button:focus, a.cf_btn:focus{
  background:#fff;
}
button:active, a.cf_btn:active{
  background:#ccc;
  border:1px solid #ccc;
}
