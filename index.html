cFlex Description

cFlex is a kind of blueprint to setup a website layout with CSS Flexbox.

The cflex3.css file is the main framework which can be used as default.

cflex3.css file has a weight at 5,7 kB / minified 1,3 kB

The cflex3_radix.css file is a basic template, with a few basic functions to get started quickly.

cflex3_radix.css file has a weight at 7,5 kB / minified 2,8 kB

The order of the Flexbox blocks can be easily adjusted.
There a some useful classes available: e.g with the class .cF_50 you can switch a section or a div to 50% width.

.cF_50{
  flex:1 1 50%;
  flex:1 1 calc( 100% / 2 );
}

Tip: Minimizes before using the CSS files so everything commented out is not exported. Only the required CSS will be loaded.

What's in?

  • normalize.css / v8.0.1
  • cflex3-name.css / v3.0.0
  • cflex3-name_radix.css / v3.0.0
  • cflex3-name.html

How to use!

  • leave normalize.css as it is
  • rename cflex3-name.css to cflex3.css
  • rename cflex3-name_radix.css to layout.css
  • rename cflex3-name.html to index.html
